# 💘 Valentine's Day Animated Card

A sweet, pure HTML and CSS animated Valentine's Day card created for someone special. 

## ✨ Features
* **No JavaScript Needed:** ขับเคลื่อนด้วยพลังของ CSS Animations ล้วนๆ (Pure CSS)
* **Floating Hearts:** อนิเมชั่นหัวใจลอยขึ้นฟ้าแบบชิลๆ ด้วย `@keyframes`
* **Secret Hover Message:** ลองเอาเมาส์ไปชี้ที่ข้อความ "Happy Valentine's Day!" ดูสิ แล้วจะเจอข้อความลับซ่อนอยู่!
* **Background Music:** มีการแอบฝัง iframe เล่นเพลงคลอเบาๆ สร้างบรรยากาศ

## 🛠️ How to Use (วิธีใช้งาน)
โปรเจกต์นี้ไม่มีอะไรซับซ้อน ไม่ต้อง Install อะไรให้วุ่นวาย:
1. ดาวน์โหลดหรือ Clone repository นี้ลงเครื่อง
2. เปิดไฟล์ `html valen.html` ด้วยเว็บเบราว์เซอร์อะไรก็ได้ (Chrome, Safari, Edge)
3. Enjoy the sweetness! 💌

## 📁 File Structure
* `html valen.html`: ไฟล์โครงสร้างหลักของการ์ดและโค้ดฝังเพลง
* `css valen v2.css`: ไฟล์สไตล์ลิ่ง ทำสีสัน รูปทรง และอนิเมชั่นทั้งหมด
